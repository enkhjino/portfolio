Happy Hikers


  • NodeJS
  • ExpressJS
  • MongoDB

This app connects hiking lovers to share, review and view hiking trails.


A comprehensive web application built using Node.js, Express.js, Javascript, and CSS. Happy Hikers is a full CRUD application with Google Authentication. It uses Mongoose to connect Node.js & MongoDB while its view engine utilizes ejs. The favorite button has functionality that keeps track of the trails that the logged user has favorited.
