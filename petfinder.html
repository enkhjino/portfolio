Paw Friends


  • MERN Stack
  • PetFinderAPI

This app was created to increase pet adoption after the high rate of post-pandemic pet abandonment.


A full stack web application built using React, Express.js, Node.js and MongoDB. The application lets users find their perfect pet through a survey they fill out according to their own criteria. It calls the PetFinder API using the survey to fetch matching pets.
