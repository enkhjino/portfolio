Concentration


  • HTML
  • CSS
  • JAVASCRIPT

A fully functional clone of the popular game Concentration.


A dynamic concentration game created in vanilla CSS, HTML, and Javascript. It uses DOM manipulation to flip two cards at a time from a deck of 24 cards while using sound effects.The game tracks scores and displays a timer to show how long it took the player to finish.
